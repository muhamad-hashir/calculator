While learning full stack web development from Scrimba, I completed the HTML and CSS portion and then started JavaScript. This calculator was my second JavaScript project.

Key concepts learned and applied:

- Functions — separate functions for each arithmetic operation
- DOM manipulation — selecting and updating elements on the page
- getElementById — targeting specific HTML elements
- .textContent — displaying results on the webpage
- Conditional statement (if/else) — handling division by zero error
- Template logic — showing error message when denominator is zero
- Flexbox — arranging numbers and buttons in rows
- Linear gradient — applying gradient background and button colors
- Border radius — rounded boxes for numbers and buttons
- Hover effect — buttons change opacity on hover

Built using HTML, CSS and JavaScript.
